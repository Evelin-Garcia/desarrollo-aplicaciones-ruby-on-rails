# Ruby Basics

## Naming Convention

### Variables 

```ruby
miVariable   # variable local 
$miVariable  # variable global
@miVariable  # variable de instancia 
@@miVariable # variable de clase
```

### Clases

```ruby
# comienza con Mayúscula
# si tiene más de 2 palabras usar CamelCase
class TarjetaCredito 

end 
```
